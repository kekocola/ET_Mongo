# AGENTS.md

## 项目概述

本仓库是 ET C# 服务器 MongoDB 配套管理工具，目标是提供 Windows 桌面端管理能力：

- 玩家数据查询
- 玩家数据导出
- 玩家数据导入
- 玩家数据批量修改
- MongoDB 服务器性能数据查看
- MongoDB 连接管理

当前阶段完成 WPF 项目初始化和 UI 框架搭建，业务逻辑与 MongoDB 真实连接后续实现。

## 技术栈

- C#
- WPF
- .NET 8，目标框架 net8.0-windows
- 暂不引入第三方 MVVM 框架，当前使用轻量级手写 ViewModel 与 Command。

## 目录约定

- src/ETMongoAdmin/：WPF 主程序工程。
- scripts/：构建、发布等辅助脚本。
- work/：本地临时分析或草稿目录，不纳入 Git。
- outputs/：Codex 交付物目录，不纳入 Git。

## 开发约定

- 每次功能开发完成后，都必须同步维护本文件，记录新增模块、运行方式、关键设计决策或注意事项。
- UI 文案默认使用中文。
- 当前 UI 先保持“可点击、可切换、可扩展”的壳结构；真实 MongoDB 连接、数据读写、导入导出等逻辑应在后续任务中逐步接入。
- 不要把连接串、账号密码、SSH 密码、私钥、导出的玩家数据等敏感信息提交到仓库。
- 修改 WPF XAML 后需要至少执行一次 scripts/build.bat 或等价的 dotnet build 验证。

## 当前 UI 框架

- 启动后左侧显示连接列表。
- 无连接时显示“暂无连接，请新增连接”的引导。
- 支持在内存中新增、编辑、删除、复制连接配置。
- 已有连接的操作不再使用独立按钮区；每个连接行右侧提供“...”菜单，菜单包含：
  - 编辑
  - 复制连接
  - 删除（需要二次确认）
- 双击某个连接后直接打开并进入工作区。
- 连接配置支持填写 MongoDB 地址、默认数据库、用户名，以及可选 SSH 代理配置：
  - SSH Host
  - SSH Port
  - SSH 用户名
  - SSH 密码（当前仅内存暂存）
  - SSH 私钥路径
- 工作区使用 Tab 分页承载：
  - 玩家数据查询
  - 玩家数据导出
  - 玩家数据导入
  - 玩家数据批量修改
  - Mongo 性能
  - 连接管理
